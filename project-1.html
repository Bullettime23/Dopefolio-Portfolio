Project Overview

ANKEY Asap is a software package for advanced analytics of information security events and incidents with behavioral analysis functions. It collects information from private network infrastructure to process, filter, group, and analyze data. ASAP provides attack chain identification to help companies enforce their network security.

Since the start of my work in the development team, I created a number of the components of the single-page application from scratch, migrated between UI kits, provided Unit tests and custom Reactive-X state management. Besides ensuring the code quality and delivering features according to business requirements, I cooperate with the design team and analytics to provide the best user experience.
